##################
Annotated Examples
##################

This section describes each of the examples provided with Chaco.  Each
example is designed to be a stand-alone demonstration of some of
Chaco's features.  Though they are simple, many of the examples have
capabilities that are difficult to find in other plotting packages.

Extensibility is a core design goal of Chaco, and many people have used the
examples as starting points for their own applications
